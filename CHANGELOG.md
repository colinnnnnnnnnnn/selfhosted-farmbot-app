# FarmBot Web App Changelog

## November 2025

#### What's new

* Added support for the Rotary Tool (back)
* Implemented Weeder Tool functionality (front & back)
* Added Seed Injector capabilities (back)
* Integrated Soil Sensor readings (back)

#### Changes

* Photo gallery changes:
  * RESTful API for photo management
  * Metadata storage in database
  * Coordinate tracking
  * Automatic thumbnail generation
  * WebSocket updates for real-time changes

* OAuth2 implementation
* Migrated to Daphne as default ASGI server
* Added Docker support with docker-compose
* Implemented WebSocket for live updates
